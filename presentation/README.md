# Presentation slides (20 slides)

## Option 1: Present from the browser

1. Open **slides.html** in Chrome or Edge (double-click or drag into browser).
2. Press **F11** for full screen (or use the browser’s full-screen option).
3. Use **Space**, **Down arrow**, or **Page Down** to go to the next slide.
4. Use **Up arrow** or **Page Up** to go back.

The deck uses a light theme (white/light green) to match A-grade dissertation examples.

---

## Option 2: Build in PowerPoint or Google Slides

Use **SLIDES_OUTLINE.md** in the project root for the full outline (titles + bullets for each slide).  
Copy the content from the outline into new slides. Keep:

- One idea per slide
- Same title and body font on every slide
- Light background, dark text
